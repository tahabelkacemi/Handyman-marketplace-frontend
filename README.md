# 🌐 KhedmaNet - Handyman Marketplace (Frontend)

A modern and responsive single-page web application (SPA) designed to connect clients with professional handymen. Built with dynamic user interfaces to ensure a seamless service-matching experience.

## 🚀 Tech Stack
* **React.js** (Component-based UI architecture)
* **Tailwind CSS** (Utility-first styling framework)
* **Axios** (HTTP client for secure API communication)

## ✨ Key Features
* **Secure System Authentication:** Full integration with backend encrypted login and signup forms.
* **Dynamic Search & Filters:** Real-time search engine to filter handymen by skill and location.
* **Star Rating System:** Interactive star rating capability with live visual state updates.
* **Role-Based Views:** Dynamic frontend conditional rendering tailored for both regular Users and Admin dashboards.
